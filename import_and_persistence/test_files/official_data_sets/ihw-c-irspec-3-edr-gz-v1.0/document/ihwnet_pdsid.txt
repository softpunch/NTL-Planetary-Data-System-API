            IHW Network to PDS DATA_SET_ID Correspondences                    
            ==============================================                    
                                                                              
This file provides a quick listing of PDS DATA_SET_ID values corresponding    
to the file prefixes used by the various International Halley Watch Nets.     
The IHW observation file names consisted of a prefix followed by a sequential 
number.  The sequence number was left-padded with zeroes so the the file name 
without extension was always eight characters long.  The nets used different  
ranges of sequence numbers for the three comets, and also to distinguish      
between comet observations and calibration observations. In the lists below,  
the first column is the file prefix used; the second is the PDS DATA_SET_ID   
value assigned to at least some of the files with that prefix; the third a    
note field.                                                                   
                                                                              
Only the original ("V1.0") data sets are listed below. Users interested in    
the actual data would be well advised to check for later versions of these    
data sets in the PDS catalog.  Data sets with the notation "no digital data"  
either record attempted observations that returned null results, or provide   
some documentation for results submitted in an analog format (photo, trace,   
etc.) that could not be digitized.  Some of these were reproduced in the      
printed IHW Archive publications.
