GEOMETRY/ Directory Contents                                                  
============================                                                  
This directory contains the ephemeris files from the HAL_0024 EPHEM/          
directory.                                                                    
                                                                              
   ephem.tab  Contains position information for Comet Giacobini-Zinner        
              in one-day increments from 1984-01-01 to 1987-06-01             
                                                                              
   ephem.txt  Explanation of the provenance and content of ephem.tab
