DOCUMENT/ Contents                                                            
==================                                                            
                                                                              
This directory contains the complete contents of the original HAL_0026        
disk DOCUMENT/ directory.                                                     
                                                                              
***** File DOCINFO.TXT                                                        
                                                                              
             Contents of the Document Directory                               
                                                                              
This file describes the content of the DOCUMENT directory and many of         
the files are listed below:                                                   
                                                                              
DOCINFO.TXT      - This file describing contents                              
CDTREE.TXT       - Layout of the disk structure                               
ACKNWLDG.TXT     - File listing contributors to this volume                   
CDSTRUCT.TXT     - Log file accumulated during the premaster process          
VOLINFO.TXT      - Discussion of data, explanatory files, index tables,       
                   and volume organization.                                   
                                                                              
Other subdirectories are located in the DOCUMENT directory as follows:        
                                                                              
ESA_1066         - digitized articles about each Mission                      
STDS_IHW         - files to outline the PDS and FITS use by IHW
