Demo 1: entities enumeration

This demo shows how to get info about datasets using getDataSetsInfo operation.
The procedure for other entities (target types, targets, missions, instruments, data files) is the same, just use another getXXXInfo operation.

Press button 1 to get list of all datasets
Press button 2 to apply pagination: items per page = 3, page number = 2
Press button 3 to apply restriction: get all datasets from the mission with id 3 (DEEP SPACE 1 mission)
